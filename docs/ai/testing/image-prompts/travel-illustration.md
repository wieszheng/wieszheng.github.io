# Role: 资深二次元插画师 & 旅游手绘专家

## Project
请结合联网搜索结果，为 `[目标目的地]` 绘制一张【吉伊卡哇 (Chiikawa) 风格的 `[N]` 天游玩路线全景图】。

## Data Workflow (数据逻辑)
1. **行程规划**：
   - 搜索该目的地最经典的 `[N]` 天行程（例如：5天4晚）。
   - 按时间顺序梳理每日核心安排：上午景点 -> 午餐特色 -> 下午景点 -> 晚餐/夜景。
   - **关键数据**：包含真实的交通耗时。

## Visual Style (画风核心 - 必须严格执行)
- **艺术风格**：**Chiikawa Style (吉伊卡哇风格)** / **Japanese Soft Doodle (日式软萌涂鸦)**。
- **线条与色彩**：使用粗且柔软的黑色轮廓线，配合低饱和度的粉彩配色（马卡龙色系），画面要极其可爱、治愈。
- **角色设定 (The Trio)**：
  - 🐰 **美食担当 (Usagi/乌萨奇)**：一只黄色的兔子，表情疯狂或兴奋，要在画面中大口吃当地美食。
  - 🐱 **导游担当 (Hachiware/小八)**：一只蓝白色的小猫（蓝色耳朵），表情温柔，手里拿着小旗子或地图指路。

## Content Layout (画面布局)
1. **游戏化地图结构**：
   - 采用**S型蜿蜒路径**，类似大富翁 (Monopoly) 游戏地图。
   - 起点是"出发"，终点是"打卡成功"。

2. **每日节点 (Day 1 - Day N)**：
   - 用**不规则的彩色气泡**包裹每一天的行程文字。
   - **文字排版**：标题用可爱的圆体字，重点信息（如价格）用醒目的红色手写体标注。

3. **场景互动**：
   - 不要只画静止的图标，要画**吉伊卡哇三小只与景点的互动**。
   - *例如：* 去爬山，Usagi 已经冲到山顶了，Chiikawa 还在半山腰哭；去吃火锅，三只围在一起流口水。

## Output Elements
- **结尾彩蛋**：在路线终点，画三只小动物举着奖杯🏆或拥抱在一起庆祝。
- **装饰细节**：背景中散落一些星星✨、爱心💖、音符🎵等日式手账装饰元素。
- **画幅**：16:9 横版高清长图。
